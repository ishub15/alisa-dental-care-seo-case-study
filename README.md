# Alisa Dental Care – SEO Case Study

## Project Overview

Alisa Dental Care is a healthcare website built using HTML and Bootstrap with more than 65 service and informational pages. As part of my SEO training and practical implementation work, I handled on-page optimization activities across the website to improve content structure, search visibility, and user experience.

---

## My Role

SEO Project Trainee | On-Page SEO Implementation

I worked on page-level optimization and SEO implementation across multiple service pages while understanding how search intent changes across healthcare topics.

---

## Website Scope

- 65+ pages optimized
- Service pages and informational pages
- HTML + Bootstrap website structure
- Healthcare / Dental niche

---

## SEO Activities Performed

### Search Intent Understanding

Before optimizing pages, I reviewed the purpose of each page and considered what users may actually search.

Examples:

- "What is a dental crown?" → informational intent
- "Best dental implant treatment" → commercial intent

This helped me avoid treating every page the same way.

---

### On-Page SEO Optimization

Implemented:

- Meta title optimization
- Meta description creation
- H1, H2, H3 structure improvements
- Primary and secondary keyword implementation
- Image alt text optimization
- Internal linking strategy
- Content structure improvements
- Canonical tag review
- Open Graph implementation
- Twitter Card support

---

## Sample Pages Optimized

- Dental Crowns
- Root Canal Treatment
- Wisdom Tooth Extraction
- Dental Implants
- Cosmetic Dentistry

---

## Challenges Faced

Many dental pages had overlapping topics and similar keywords. Instead of repeating the same optimization process across all pages, I focused on differentiating content based on search intent and page purpose.
