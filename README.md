# Alisa Dental Care SEO Case Study

## Project Overview

SEO project completed during internship/training involving a dental clinic website built using HTML + Bootstrap.

### Website Scope

- 65+ pages optimized
- Dental service pages
- Informational healthcare content

## SEO Activities Performed

- Meta title optimization
- Meta description creation
- H1, H2, H3 optimization
- Keyword placement
- Internal linking
- Image alt text optimization
- Canonical review
- Open Graph support
- Twitter Card support
- Content structure optimization

## Sample Pages Optimized

- Dental Crowns
- Wisdom Tooth Extraction
- Root Canal Treatment
- Dental Implants
- Cosmetic Dentistry

## Key Learnings

- Large website optimization workflow
- Metadata implementation
- Search intent mapping
- On-page SEO execution
- Internal linking strategy
